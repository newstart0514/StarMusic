# cloudmusic

一个基于网易云API开发的音乐平台，使用请配合[网易云api](https://github.com/Binaryify/NeteaseCloudMusicApi)使用。

## Project setup
```
npm install
or
yarn
```

### Compiles and hot-reloads for development
```
npm run serve
or
yarn serve
```

### Compiles and minifies for production
```
npm run build
or
yarn build
```
