# cloudmusic

一个基于网易云API开发的音乐平台，使用请配合[网易云api](https://github.com/Binaryify/NeteaseCloudMusicApi)使用。

## 安装项目依赖
```
npm install
or
yarn
```

### 编译并开发（或者预览）该项目
```
npm run serve
or
yarn serve
```

### 编译并打包该项目
```
npm run build
or
yarn build
```
